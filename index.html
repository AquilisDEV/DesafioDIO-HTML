Diferenças entre Client & Server

Voltar

Client

O termo "Client" como proprio diz, é o Cliente, no qual é usado para definir tudo aquilo visto no computador/celular do cliente, seja imagens, áudios, vídeos e textos.

Sua parte trabalhada é definida pelo Front-end, onde o desenvolvedor, desenvolve o "rosto" da página web utilizando as linguagens: HTML, CSS e JS. Enquanto o Back-end cuida dos dados de baixos dos panos onde o cliente (Usuário final) não pode ver.

Server

O termo "Server" significa Servidor, ou seja, aquele que serve os serviços de dados para página web, como armazenamento de dados como logins, passwords, dados do cartão ao efetuar uma compra ou mesmo seus arquivos no caso de um site que armazene perfis como Facebook, Instagran, Twitter e outros.

Sua parte trabalhada é definida pelo Back-end, onde o desenvolvedor cria um banco de dados utilizando Python, JavaScript, SQL e PHP.

Oque é HTML?

Antes de tudo precisamos entender que HTML, não é uma uma linguagem de programção.

HTML significa, HyperText Markup Language (Linguagem de marcação de Hipertexto). Nada mais é do que um componente web para criação de páginas web utilizando, textos, imagens, videos e audios.

Tags

Como na propria descrição de HTML, as tags são marcações de texto usados para identificar e dizer ao navegador como interagir com X elemento desiginado na página, como por exemplo:

As tags "h1 até h6" (headings) são usados para definir títulos na página, tendo cada uma seu nível de importância.

Atributos

Atributos são palavras especiais usadas para modificar ou categorizar para idenficar um elemento HTML seja para ser usado futuramente na formatação usando CSS ou para identificar e facilitar o trabalho da implementação de interações e etc com o JavaScript, como por exemplo:

"header class="cabecalho"

O atributo Class pode ser usado após uma tag para dar a ela uma Classificação que pode ser usada posteriormente no CSS para formatações.

Git e Github

Voltar

Git

Git é uma ferramenta para controlar seu código durante um trabalho/projeto, ele salva seu código apartir de um momento e o salva para que caso haja um problema no código você possa pegar uma versão anterior dele para que possa trabalhar sem os conflitos anteriores.

Github

Github é um site onde você pode guardar seus códigos, projetos. É principalmente usado como ferramenta de portifólio onde pessoas e empresas podem ver seu nível como programador ao acessar seus repositórios
